## Backend framework

**Status:** Open — Express assumed in scaffold but not committed.

Interested in trying **ElysiaJS** as the Core API server instead of Express. ElysiaJS is Bun-native, TypeScript-first, and has built-in validation via its type system (which might reduce or replace the zod layer in the API routes). Worth evaluating before scaffolding `server/`.

Reference: https://elysiajs.com
